# tmux Session Managers

There are a number of `tmux` session managers. The recommened one to use is `tmuxp`. 

**Session Managers**

- [tmuxp](https://github.com/tmux-python/tmuxp) (RECOMMEND)
  - Implemented in `python`
  - [Tmuxp - Archlinux Wiki](https://wiki.archlinux.org/title/Tmuxp)
- [tmuxinator](https://github.com/tmuxinator/tmuxinator)
  - Implemented in `ruby`
- [teamocil](https://github.com/remi/teamocil)
  - Implemented in `ruby`
